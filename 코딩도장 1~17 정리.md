8-2. 논리 연산자-  and, or, not

 	1) and :  두 값이 모두 True어야 True

 	2) or : 두 값 중 하나라도 True면 True

 	3) not : 논리값 뒤집음. not True는 False가 되고, not False는 True.

    -  and, or, not 논리 연산자가 식 하나에 들어있으면 not, and, or 순으로 판단

 	- not True and False or not False

 		False and False or True

 		False or True

 		True

 	- 비교 연산자(is, is not, ==, !=, <, >, <=, >=)를 먼저 판단 후 논리 연산자(not, and, or)를 판단

    - 정수, 실수, 문자열을 불로 만들 때는 bool을 사용하면 됩니다. 이때 정수 1은 True, 0은 False ( 정수 0, 실수 0.0 이외 모든 숫자는 True)

    -  문자열의 내용 자체는 판단하지 않으며 값이 있으면 True( 빈 문자열 " " 제외)

 	  (1) 논리 연산자는 마지막으로 단락 평가를 실시한 값을 그대로 반환

 		ex.  >>> True and 'Python'

 				  'Python'

 				>>> 'Python' and False

 				False

      (2) 여기서는 문자열 'Python'을 True로 쳐서 and 연산자가 두 번째 값까지 확인하므로 두 번째 값이 반환됨. 만약 다음과 같이 and 연산자 앞에 False나 False로 치는 값이 와서 첫 번	째 	    값 만으로 결과가 결정나는 경우에는 첫 번째 값이 반환

 		>>> False and 'Python'

 		False

 		>>> 0 and 'Python'    # 0은 False이므로 and 연산자는 두 번째 값을 평가하지 않음

 		0

     (3) or 연산자에서 첫 번째 값만으로 결과가 결정되므로 첫 번째 값이 반환. 두 번째 값까지 판단해야 한다면 두 번째 값이 반환.

 		>>> 'Python' or True

 		'Python'

 		>>> False or 'Python'

 		'Python'
